## Getting Started

Clone the repository:

```bash
git clone https://github.com/seizmeia/seizmeia.git
```

To run the application:

```bash
make bup
```

To see it work:

```bash
curl localhost:8000
```

The application will autoreload on `/seizmeia` changes.

To stop the application:

```bash
make down
```
